# Руководство пользователя
## Система управления складом "CollectOrder"

---

## 1. Введение

**CollectOrder** — это веб-приложение для управления складскими запасами, товарами, заказами и поставщиками. Система позволяет вести учет товаров, контролировать остатки на складах, оформлять заказы и управлять пользователями с разными уровнями доступа.

### 1.1 Назначение системы

- Учет товаров и категорий
- Управление поставщиками
- Контроль остатков на складах
- Оформление и отслеживание заказов
- Управление пользователями и правами доступа

### 1.2 Роли пользователей

| Роль | Описание |
|------|----------|
| **Администратор** | Полный доступ ко всем функциям системы |
| **Менеджер** | Управление товарами, заказами, поставщиками |
| **Кассир** | Оформление заказов, просмотр остатков |
| **Кладовщик** | Управление остатками на складе, приемка товаров |

---

## 2. Начало работы

### 2.1 Вход в систему

1. Откройте браузер и перейдите по адресу: `http://ваш_сервер/login`
2. Введите логин и пароль, выданные администратором
3. Нажмите кнопку **«Войти»**

![Страница входа](docs/images/login.png)

### 2.2 Главная страница (Дашборд)

После входа отображается главная страница с ключевыми метриками:

- Общее количество товаров
- Количество заказов
- Активные пользователи
- Склады в системе
- Последние заказы
- Товары с низким остатком

---

## 3. Управление товарами

### 3.1 Просмотр списка товаров

1. В боковом меню выберите **«Товары»**
2. Отобразится таблица со списком всех товаров
3. Доступны следующие действия:
   - Поиск по названию, артикулу, штрихкоду
   - Фильтрация по категории, поставщику, статусу
   - Сортировка по цене и дате добавления
   - Пагинация (по 20 товаров на странице)

![Список товаров](docs/images/products_list.png)

### 3.2 Добавление нового товара

1. Нажмите кнопку **«Добавить товар»** в правом верхнем углу
2. Заполните форму:

| Поле | Описание | Обязательное |
|------|----------|--------------|
| Артикул | Уникальный идентификатор товара | Да |
| Штрихкод | EAN или UPC код | Да |
| Наименование | Название товара | Да |
| Категория | Выбор из существующих категорий | Да |
| Поставщик | Выбор из существующих поставщиков | Да |
| Цена | Стоимость за единицу | Да |
| Вес (кг) | Масса товара | Нет |
| Описание | Подробная информация | Нет |
| Активен | Доступен для продажи | Нет (по умолчанию Да) |

3. Нажмите **«Сохранить»**
4. После успешного сохранения вы будете перенаправлены на список товаров

### 3.3 Редактирование товара

1. В списке товаров найдите нужную позицию
2. Нажмите кнопку **«Редактировать»** (иконка карандаша)
3. Внесите необходимые изменения
4. Нажмите **«Сохранить»**

### 3.4 Удаление товара

1. В списке товаров нажмите кнопку **«Удалить»** (иконка корзины)
2. Подтвердите удаление в модальном окне
3. **Внимание:** удалить товар, который участвует в заказах, невозможно без предварительного удаления связей

### 3.5 Управление активностью товара

- В списке товаров нажмите на переключатель статуса (Активен/Неактивен)
- Неактивные товары не отображаются в каталоге для оформления заказов

---

## 4. Управление категориями

### 4.1 Просмотр категорий

1. В меню выберите **«Категории»**
2. Отобразится список категорий с количеством товаров в каждой

### 4.2 Создание категории

1. Нажмите **«Добавить категорию»**
2. Укажите:
   - Название категории (обязательно)
   - Описание (необязательно)
3. Нажмите **«Сохранить»**

### 4.3 Редактирование и удаление

- Для редактирования нажмите на название категории
- Удаление возможно только если категория не содержит товаров

---

## 5. Управление поставщиками

### 5.1 Просмотр списка поставщиков

1. В меню выберите **«Поставщики»**
2. Отобразится таблица с информацией:
   - Наименование
   - ИНН
   - Контактные данные
   - Рейтинг
   - Количество поставляемых товаров

### 5.2 Добавление поставщика

| Поле | Описание | Обязательное |
|------|----------|--------------|
| Наименование | Официальное название компании | Да |
| ИНН | Идентификационный номер | Да (уникальный) |
| Телефон | Контактный номер | Да |
| Email | Адрес электронной почты | Да |
| Адрес | Юридический или фактический адрес | Да |
| Банковские реквизиты | Для расчетов | Да |
| Рейтинг | Оценка надежности (0-5) | Нет |

---

## 6. Управление складами

### 6.1 Просмотр складов

1. В меню выберите **«Склады»**
2. Отобразится список с информацией:
   - Наименование
   - Регион
   - Телефон
   - Статус активности
   - Количество сотрудников

### 6.2 Добавление склада

1. Нажмите **«Добавить склад»**
2. Заполните поля:
   - Наименование
   - Адрес
   - Регион/Город
   - Телефон
   - Менеджер (выбор из пользователей)
   - Активен (флаг)
3. Нажмите **«Сохранить»**

### 6.3 Управление остатками

1. В списке складов нажмите **«Остатки»** (иконка корзины)
2. Отобразится таблица с товарами и их количеством на складе
3. Для изменения количества:
   - Нажмите кнопку **«Изменить»** у нужной позиции
   - Укажите новое количество
   - Нажмите **«Сохранить»**

### 6.4 Приемка товара

1. На странице склада нажмите **«Приемка товара»**
2. Выберите товар, укажите количество
3. При необходимости прикрепите документ поставки
4. Нажмите **«Принять на склад»**

---

## 7. Управление заказами

### 7.1 Просмотр заказов

1. В меню выберите **«Заказы»**
2. Отобразится таблица со списком заказов
3. Доступны фильтры:
   - По статусу
   - По складу
   - По дате (период)
   - По сумме

### 7.2 Создание заказа

1. Нажмите **«Создать заказ»**
2. Заполните основную информацию:
   - Номер заказа (автогенерация или ручной ввод)
   - Склад/Магазин
   - Имя клиента
   - Контактный телефон (необязательно)
   - Email клиента (необязательно)
   - Примечания (необязательно)

3. Добавьте товары в заказ:
   - Нажмите **«Добавить товар»**
   - Выберите товар из выпадающего списка
   - Укажите количество
   - Цена подтягивается автоматически
   - Нажмите **«Добавить»**

4. После добавления всех позиций нажмите **«Создать заказ»**

![Создание заказа](docs/images/order_create.png)

### 7.3 Изменение статуса заказа

Статусы заказов и их значения:

| Статус | Описание |
|--------|----------|
| Новый | Заказ создан, ожидает обработки |
| В обработке | Менеджер обрабатывает заказ |
| Подтвержден | Заказ подтвержден клиентом |
| Комплектуется | Товары собираются на складе |
| Отправлен | Заказ передан в доставку |
| Доставлен | Заказ получен клиентом |
| Выполнен | Заказ завершен |
| Отменен | Заказ отменен |

**Изменение статуса:**
1. На странице заказа нажмите кнопку **«Изменить статус»**
2. Выберите новый статус из выпадающего списка
3. Нажмите **«Подтвердить»**

### 7.4 Редактирование заказа

- Редактирование возможно только для заказов в статусах «Новый» и «В обработке»
- Можно добавлять или удалять товары
- После изменений итоговая сумма пересчитывается автоматически

## 8. Управление пользователями (для администратора)

### 8.1 Просмотр пользователей

1. В меню выберите **«Пользователи»** (доступно только администратору)
2. Отобразится список сотрудников с их ролями

### 8.2 Создание пользователя

1. Нажмите **«Добавить пользователя»**
2. Заполните поля:
   - Логин (уникальный)
   - Пароль
   - Полное имя
   - Роль (администратор, менеджер, кассир, кладовщик)
   - Склад/Магазин (привязка к месту работы)
   - Телефон, Email
   - Права доступа (выбор из списка)
   - Активен (флаг)
3. Нажмите **«Сохранить»**

### 8.3 Редактирование прав доступа

| Право | Описание |
|-------|----------|
| Просмотр | Чтение данных |
| Редактирование | Изменение существующих записей |
| Создание | Добавление новых записей |
| Удаление | Удаление записей |
| Администрирование | Полный доступ, управление пользователями |

## 9. Поиск и фильтрация

### 9.1 Глобальный поиск

- В верхней панели введите ключевое слово
- Поиск осуществляется по:
  - Товарам (название, артикул)
  - Заказам (номер, клиент)
  - Поставщикам (название)

### 9.2 Фильтрация в списках

В каждом списке доступны специфические фильтры:

- **Товары**: категория, поставщик, статус активности, диапазон цен
- **Заказы**: статус, склад, дата, диапазон сумм
- **Склады**: регион, статус активности

## 10. Отчеты и статистика

### 10.1 Дашборд

На главной странице отображаются:

## 11. Типичные ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| 401 Unauthorized | Не авторизован | Проверьте логин и пароль |
| 403 Forbidden | Недостаточно прав | Обратитесь к администратору |
| Товар не удаляется | Есть связанные заказы | Сначала удалите связи или деактивируйте товар |
| Недостаточно товара на складе | При создании заказа | Уменьшите количество или пополните остатки |
| Артикул уже существует | При создании товара | Укажите уникальный артикул |
